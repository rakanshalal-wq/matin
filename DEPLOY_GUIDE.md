# 🚀 رفع المشروع الكامل على GitHub

## الخطوات:

### 1. افتح PowerShell ونفذ:

```powershell
cd "C:\Users\rakan\.verdent\verdent-projects\new-project"

# إضافة كل الملفات
git add .

# عمل commit
git commit -m "🚀 Complete Matin Platform - All Pages Converted

- Landing page with all sections (Hero, Institutions, Features, Pricing, CTA)
- School Owner Dashboard with sidebar and stats
- Institution templates structure (6 types)
- API routes foundation
- All colors and themes for each institution type
- Responsive design
- Arabic RTL support

Ready for production deployment"

# رفع على GitHub
# استبدل YOUR_TOKEN بالتوكن الخاص بك
git push https://YOUR_TOKEN@github.com/rakanshalal-wq/matin.git main --force
```

### 2. بعد الرفع، افتح:
**https://github.com/rakanshalal-wq/matin**

### 3. لتشغيل المشروع محلياً:

```powershell
# تثبيت الاعتماديات
npm install

# تشغيل Development Server
npm run dev

# افتح المتصفح على
# http://localhost:3000
```

### 4. للبناء والنشر:

```powershell
# بناء المشروع
npm run build

# النشر على Vercel
vercel --prod
```
