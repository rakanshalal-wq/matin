5. الإصلاحات الأمنية

  • ✓ JWT التحقق الكامل من التوقيع عبر WebCrypto HMAC-SHA256
  • ✓ RBAC - التحكم بالوصول حسب الدور (super_admin, owner, admin, teacher)
  • ✓ globalThis pool reuse - منع تسريب اتصالات PostgreSQL
  • ✓ HSTS - فرض HTTPS (max-age=63072000)
  • ✓ Rate Limiting على login (burst=10) و API (burst=50)
  • ✓ حظر الوصول لـ .git
  • ✓ حماية uploads من الملفات التنفيذية
  • ✓ X-Frame-Options: DENY + X-Content-Type-Options: nosniff
